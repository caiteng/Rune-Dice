# 猫猫守夜

《猫猫守夜》是一款竖屏猫猫题材幸存者小游戏，主平台为抖音小游戏，使用 Godot 4.5 开发。

项目目标：

- 以抖音小游戏上线和广告变现为主目标。
- 保留 Web 自动部署试玩能力，方便快速反馈。
- 使用 Godot + GDScript，降低后续阅读和维护成本。
- 第一版避免复杂角色动画，使用静态图标、简单动效、数字反馈和清晰 UI。

当前阶段：立项与技术样板准备。

## 核心方向

- 类型：竖屏轻量幸存者
- 题材：猫猫守夜，清理夜晚房间里的小幽灵、灰尘怪、噪音球和梦魇
- 操作：下半屏浮动虚拟摇杆移动，武器自动攻击
- 单局：MVP 先做 6 分钟局，后续扩展 10 分钟标准局
- 成长：局内升级三选一，武器满级后通过宝箱进化
- 变现：复活、刷新升级、双倍结算、局外补给

## 平台策略

抖音小游戏是正式产品；Web 是开发试玩版。

Web 版用于验证手感、节奏、UI、数值和内容反馈。广告、抖音 API、真机适配和包体表现必须以抖音开发者工具和真机预览为准。

## 技术原则

- 主版本使用 Godot 4.5。
- 只使用 GDScript。
- 不使用复杂角色帧动画。
- 不做大地图相机玩法。
- 不依赖后端完成 MVP。
- 平台能力通过适配层封装，避免业务代码直接散落平台 API。
