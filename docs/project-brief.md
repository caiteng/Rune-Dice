# 项目立项：猫猫守夜

## 一句话定位

《猫猫守夜》是一款竖屏猫猫题材幸存者小游戏。玩家用下半屏浮动虚拟摇杆控制猫猫移动，武器自动攻击夜晚出现的怪物，通过升级和武器进化坚持到天亮。

## 项目目标

- 做出一款可上线抖音小游戏的轻量幸存者产品。
- 用成熟幸存者类玩法节奏作为第一版基准，优先保证好玩。
- 通过广告点实现基础变现验证。
- 保留 Web 自动部署试玩流程，降低反馈成本。

## 平台与技术

- 引擎：Godot 4.5
- 语言：GDScript
- 主平台：抖音小游戏
- 预览平台：Web
- 屏幕方向：竖屏
- 目标操作：单手浮动虚拟摇杆

## 设计约束

- 不做复杂角色动画。
- 不做实时小队战斗。
- 不做复杂相机和大地图。
- 不做多人、账号、后端排行。
- MVP 不做内购。
- 广告必须由玩家主动触发，不能误触或自动弹出。

## 视觉方向

可爱、夜晚、柔和，但不要过度依赖美术资产。

核心表现用：

- 静态猫猫图标或轻量骨架表现
- 简单怪物图标
- 血条、伤害数字、闪白、抖动
- 武器轨迹、光圈、区域提示
- 宝箱、升级卡片、结算面板

## 商业化点

- 死亡后看广告复活一次
- 升级三选一看广告刷新
- 结算看广告双倍小鱼干
- 局外看广告领取夜宵补给

## 当前优先级

第一版先全力赶出可试玩版本。等项目有一套可复用的成功经验后，再另行规划对外内容。
